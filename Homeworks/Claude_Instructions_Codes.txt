2. NUMERICAL CORE — SEM CONVENTIONS
────────────────────────────────────────────────────────────────────────────────

• Quadrature: always use xwlgl(nln) — LGL nodes on [-1,1]
• Reference element: [-1,1]  (NOT [0,1] as in FEM/HMZ_1D)
• Jacobian: BJ = 0.5*(x1-x0),  x_phys = 0.5*(x1+x0) + BJ*xi
• Basis functions: basis_and_der_at_lgl(nodes_1D, nln)
• nln = Data.p + 1  (local nodes = polynomial degree + 1)
• NEVER use ShapeBasis / EvalShapeBasis / Quadrature in new SEM code
• NEVER call xwcg, xwcgl, xwlg directly — always use xwlgl

• Time integration: θ-method with θ = 0.5 (Crank-Nicolson)
  (M + θ·dt·S) U^{k+1} = (M - (1-θ)·dt·S) U^k
• Periodic BC: identify first and last DOF in both η and q blocks

6. MainSWE.m — FUNCTION SIGNATURE AND STEPS
────────────────────────────────────────────────────────────────────────────────

  function [Errors, Solutions, Femregion, Data] = MainSWE(Data, nEl)

  Internal steps (in order):
    1.  CreateMesh(Data, nEl)
    2.  CreateFemregion(Data, Region)
    3.  Matrix1D → M (mass), B (discrete ∂_x coupling)
    4.  Build global block system:
          M_glob = blkdiag(M, M)
          S_glob = [0, B; Data.g*Data.H * B^T, 0]
    5.  Apply periodic BC to M_glob and S_glob
    6.  U0 = [eta0(x); q0(x)]
    7.  θ-method loop:
          LHS     = M_glob + theta*dt*S_glob
          RHS_mat = M_glob - (1-theta)*dt*S_glob
          for each step: U = LHS \ (RHS_mat * U)
    8.  PostProcessing(Data, Femregion, U_eta_final)
    9.  ComputeErrors if Data.calc_errors

  NEVER plots inside this function.

9. HW2 PHYSICS REMINDER
────────────────────────────────────────────────────────────────────────────────

PDE (linearized shallow water):
  η_t + q_x = 0          (mass conservation)
  q_t + gH η_x = 0       (momentum)

Domain: (0,L)×(0,T),  periodic BC: η(0,t)=η(L,t),  q(0,t)=q(L,t)

Exact solution — right-traveling wave:
  η(x,t) = η₀(x - c·t)
  q(x,t) = c · η(x,t)
  c = sqrt(gH)

Reference parameters:
  L=1,  H=1,  g=9.81,  c=sqrt(9.81),  T=0.5
  η(x,0) = exp(-50·(x-0.5)²)
  q(x,0) = c·η(x,0)

Block system (U = [η; q]):
  M_glob · dU/dt + S_glob · U = 0
  M_glob = blkdiag(M, M)
  S_glob = [0, B; gH·B^T, 0]   where B = SEM discretization of ∂_x

θ-method (θ=0.5, Crank-Nicolson):
  (M_glob + θ·dt·S_glob) U^{k+1} = (M_glob - (1-θ)·dt·S_glob) U^k

Energy (conserved for θ=0.5, no friction):
  E(t) = ∫₀ᴸ [ ½·g·η² + ½·q²/H ] dx

Expected convergence (SEM degree p):
  L² error in η:  O(h^{p+1})
